## 基于 Cloudflare Pages 部署，2024-04-05

- [官方文档](https://developers.cloudflare.com/pages/framework-guides/deploy-anything/)
- 把域名转到 Cloudflare
- 在设置里配置自定义域名
- 仓库每次提交都会触发自动部署

## 如果不想用 Cloudflare Pages

### 自动添加 .html

对于 quartz4，需要在 content 目录下放置 vercel.json，用于在点击 Wikilink 时自动添加 .html

```json
{
  "routes": [
    { "handle": "filesystem" },
    { "src": "/(.*)", "dest": "/$1.html" }
  ]
}
```
