## 现象

- 文档在 [[2023-05-12]] 0 点创建，发布失败，改成 2023-05-11，发布成功
- 笔记出现在搜索结果，点击自动跳转到 404 页面

## 猜测

- GitHub Action 服务器时间比文档头标记的创建时间早，所以未发布成功
- [[2023-05-12]] 08:00 尝试发布成功

## 解决方法

- 使用 [[ISO8601]] 格式的日期
